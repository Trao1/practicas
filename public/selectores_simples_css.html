SELECTORES SIMPLES CSS

Los selectores en css lo entenderemos como un identificador para una o un grupo de etiquetas html, el cual va a permitirnos darle estilos y poder crear una pagina web llamativa.


Existen varios tipos de selectores.


  • Selectores simple
  • Selectores compuestos
  • Selectores complejos

Selectores simples

Este tipo de selectores lo forman aquellos que solo tienen una cadena contextual, sin niguna combinacion. Para empezar se veran los selectores de "etiqueta", "clase", "id". 

<< etiqueta, clase, id >> {
    color:red;
}

Selector de etiqueta

Este tipo de selector hace referencia a las etiqueta html 

<!-- HTML -->
<p>Texto en una etiqueta p</p>
/* CSS */
p {
  color:red;
}
RESULTADO
              
Texto en una etiqueta p

Selector de clases

Este tipo de selectores hace referencia a un valor del atributo class en etiquetal html, estas comienzan con un punto concatenada al valor.

<!-- HTML -->
<p>Texto que no posee el atributo class</p>
<p class="miClase">Texto que posee el atributo class con valor "miClase"</p>

En css seria punto concatenado con el valor del atributo class -> .miClase

/* CSS */
.miClase{
  color:red;
}
RESULTADO

Texto que no posee el atributo class
Texto que posee el atributo class con valor "miClase"

Selector de id

Este selector hace referencia a un valor del atributo id en etiquetas html, es unico por lo que se usa para identificas secciones, el valor debe comentar con # concatenado al valor del atributo id.

<!-- HTML -->
<div id="miId">
  <p>Este texto sera color verde </p>
</div>

En css seria # concatenado con el valor del atributo class -> #miId

/* CSS */
#miId{
  color:green;
}
RESULTADO
              
Este texto sera de color verde
